## Описание сборки и запуска

```
mkdir build
cd build
cmake ..
make
./Compiler --src ../examples/example[1-4].in
```

Посмотреть на выхлоп работы визиторов (2-3 чекпоинты):
```
cat PrintVisitor_output
cat Interpreter_output
```

4 чекпоинт:
```
cat PrintVisitor_output
cat SymbolTreeVisitor_output
```
